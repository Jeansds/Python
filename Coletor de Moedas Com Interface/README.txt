Executavel:
Coletor_Moedas_Com_Interface

Objetivo:
Coletar as cotações de determinadas moedas de uma data até a data atual,para obter mais codigos de moedas basta acessar ao site:https://www.bcb.gov.br/estabilidadefinanceira/historicocotacoes
clicar com o botão direito em cima do campo Moeda, Inspecionar, a lista select e adicionar o valor da moeda na variavel Codigos.
